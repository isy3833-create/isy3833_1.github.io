수업시간에 만든 암호 해독 코드
코드 설명 
            cipher = input().strip()
            target = "hanshin university"

            x = int(input("x:"))
            code = int(input("code:"))

            if target == cipher[x::code]:
                print("발견")
                exit
            else:
                print("입력받은 x와 code가 잘못됨")
                for x in range(len(cipher)):
                    for code in range(-20, 21):
                        if code == 0: 
                            continue

                        if target == cipher[x::code]:
                            print("x =", x, "code =", code)
                            exit
cipher 문자열에서 시작 위치 x로부터 code의 배수 번째 글자만 진짜 암호이며, 시작 위치 x와 code가 20를 넘지 않는다는 조건일때

cipher와 x, code를 입력받고 입력받은 x와 code로 실행했을때 hanshin university가 나오지 않는다면 x와 code를 찾게 만든 코드입니다.
